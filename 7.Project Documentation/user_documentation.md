

## 🖥️ Operational Execution Manual

### Prerequisites
Ensure your local machine has Python 3.10+ installed along with access to a command-line terminal environment.

### Deployment Instructions
1. Run the application launch script from your root directory terminal:
   ```bash
   streamlit run "5. Project Development Phase/app.py"
   ```
2. Open your local browser to the web address printed in your terminal (usually `http://localhost:8501`).
3. Choose a reference concept from the available library list.
4. Drag and drop your recorded `.wav` file into the upload window panel.
5. Click the analyze button to calculate results, view data plots, and save your compiled PDF report locally.
